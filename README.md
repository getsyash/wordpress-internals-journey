# WordPress Internals Journey

Welcome to **WordPress Internals Journey**! This repository is dedicated to track my exploration of the inner workings of WordPress, understanding its core structure, and mastering how the platform operates behind the scenes.  


---

## Goals

- Understand WordPress **core architecture**.
- Explore key folders like `wp-admin`, `wp-includes`, and `wp-content`.
- Learn how WordPress **hooks, actions, and filters** work internally.
- Study **database structure** and how WordPress interacts with it.
- Experiment with **custom plugins and themes** at a core level.
- Contribute to WordPress **core, plugins, or open-source projects**.
